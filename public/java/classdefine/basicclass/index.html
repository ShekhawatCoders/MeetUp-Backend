Class :

Class defines property and behaviors of an object.It's a templete or blueprint for an object that defines what an object's datafields and methods will be .

Object :

Object is an instance of a class.The terms object and instance are interchangable.

Construcotrs :

  • Constructors should have same name as Class.
  • Constructors can not have return type even void.
  • Constructors are invoked using new operator.
  • Constructors can be overloaded , multiple constructors can have same name but different signature.
  • A class may be defined without construtors.In this case a public no-argument constructor with an empty body implicitly defined by the class.This constructor called default constructor is provided if no constructors are defined explicitly.

    Points :

  • In short, Classes are definitions for objects and objects are created from class.
  • The difference b/w classes and objects are same as apple-pie-recepe and apple-pies.You can make as many as you want apple-pie using single apple-pie-recepe.
  • A class provides a special type of methods known as constructor , they are invoked every time when we create a object of that class , if we don't define constructor jvm(java virtual machine) will automatically create this for us, in constructor we usually initialize our variables.

class Circle {
	int radius;
	//no-argument constructor
	Circle () {
		this.radius = 1;
	}
	//single parameter constructor
	Circle (int radius) {
		this.radius = radius;
	}
	public int getArea() {
		return (Math.PI*radius*radius);
	}
	public int getPerimeter() {
		return (2*Math.PI*radius);
	}
	public void setRadius(int radius) {
		this.radius = radius;
	}
}
class CircleTest {
	public static void main(String[] args) {
		Circle c1 = new Cirlce();//no-argument constructor will be called
		System.out.println(c1.getArea()); // Output : 3.14
		System.out.println(c1.radius); // Output : 1 (default radius is 1)
		Circle c2 = new Circle(2);//Single parameter constructor called
		System.out.println(c2.getArea()); // Output : 12.56
		c2.setRadius(5); // radius is now set to 5 for c2
		System.out.println(c2.radius); // Output : 5 (radius is 5)
	}
}

Important :

You can put two or more classes in one file but only one class can be public and file name will be same as that public class.Each class is compiled into a .class file so when you compile above program you will get two file "Circle.class" and "CircleTest.class" seperately.
