Overloading vs Overriding

Overloading :

Overloading is term used when two or methods have same name but differ in type or number of arguments.

Example :

public int square(int l) { }

public double square(double l) { }

Overriding :

Overriding however occures when a method shares same name and signature from its superclass.

Example : 

public  class shape {

 	public int getArea(int l,int b) {

 		return l*b;

 	}

}

public class rectangle extends shape {

		public int getArea(int l,int b) {

			return l*b;

		}

}
Here rectangle class is overriding getArea() method from its superclass.
